## Minimal html code

```html
<!DOCTYPE html>
<html>
  <head>
    <!-- Page header -->
    <meta charset="utf-8" />
    <title>Titre</title>
  </head>

  <body>
    <!-- page body -->
  </body>
</html>
```

|                 & | &amp;                          |

## Adding custom icons, CSS and JavaScript to HTML

```html
<link rel="icon" href="favicon.ico" type="image/x-icon" />
<link rel="stylesheet" href="my-css-file.css" />
<script src="my-js-file.js" defer></script>
```
